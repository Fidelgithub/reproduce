# qimai-front-end-designpro

## Getting Started

Install dependencies,

```bash
$ npm run init
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```

Publish library via `lerna publish`,

```bash
$ npm run pub
```

Deploy docs

```bash
$ npm run deploy
```
